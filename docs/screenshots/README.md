# Screenshots

These images are referenced by the root [`README.md`](../../README.md) in the
Screenshots section. Add them here with the exact filenames below so the README
renders correctly.

| File | What it should show |
|---|---|
| `editor.png` | The markdown editor with a real note open: wikilinks, a callout or math block, and the property/frontmatter editor visible. |
| `graph.png` | The global graph view: typed nodes, backlink edges, and dashed soft-reference (mention) edges, ideally with a theme applied. |
| `database.png` | A database view (table or board) driven by typed object properties, with the view menu or column picker visible. |
| `empty-state.png` | The branded empty state shown when no note is open (the `EmptyView` primitive). |

Guidelines:

- Capture on a clean sample vault (`node scripts/seed-vault.mjs ./sample-vault`).
- Use a consistent window size and theme across all four shots.
- Prefer 2x / retina captures, then keep file sizes reasonable (PNG, ideally
  under ~500 KB each).
