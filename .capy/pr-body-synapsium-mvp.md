# Synapsium MVP — Second brain veloce (Web + Desktop)

Questo PR introduce l’MVP di Synapsium con architettura condivisa Web e Desktop (Tauri), vault Markdown con YAML frontmatter, editor con anteprima e grafo delle connessioni.

## Cosa include

- App React + Vite + TypeScript con stato su Zustand
- Editor Markdown (CodeMirror 6) con autocompletamento `[[...]]`
- Frontmatter YAML per categorie predefinite (film, libri, serie_tv, video_youtube, live_event, persona, idea)
- Preview card con copertina, voto, metadati principali
- Viste: Gallery, Tabella, Timeline
- Grafo delle note (Cytoscape + dagre)
- Temi e brand: colori primario/secondario configurabili da Impostazioni
- Wrapper Tauri v1 per build desktop (Windows/macOS/Linux)
- README completo in italiano con guida e requisiti

## Perché

- Performance e footprint: Tauri consente un’app desktop leggera e veloce, condividendo il codice web
- Dati aperti: Markdown + YAML garantiscono portabilità e proprietà dei dati
- Espandibilità: importers (TMDB/Open Library/YouTube) e ricerca avanzata pronti come punti di estensione

## Impatto

- Fornisce una base solida per iterare rapidamente su import automatici, ricerca per metadati, esportazione ZIP e statistiche
- Consente agli utenti di iniziare subito ad archiviare e collegare contenuti con un’esperienza fluida
